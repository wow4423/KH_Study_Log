# KH Study Log

AWS 클라우드 기반 DevOps 개발자 양성 과정 7회차에서 학습한 내용을 17개 정규 카테고리와 PLUS LAB의 자습용 교재로 다시 구성한 정적 웹 페이지입니다.

## 구성

- HTML/CSS, JavaScript, React
- Java, JDBC, Servlet/JSP, MyBatis
- SQL, Spring MVC/Core, JPA, Spring Security
- Git, AWS, Linux, Docker, WebSocket, Redis
- 프로젝트 시작 가이드

## 로컬 실행

```bash
python -m http.server 4321
```

브라우저에서 `http://localhost:4321`로 접속합니다.

## 특징

- 원본 필기를 그대로 출력하지 않고 정의·원리·예제·비교·주의점·요약 구조로 재집필
- 116개 일자별 원문 노트를 전수 대조해 144개 독립 학습 단원으로 재구성
- 과제·제출·면접 안내는 제외하고 프로젝트 노트의 기술 지식은 해당 개념 단원에 통합
- 17개 정규 대분류와 별도 PLUS LAB, 고정 학습 목차
- 핵심 개념 표시와 반응형 레이아웃
- 수업에서 사용한 16개 프로그램의 공식 다운로드 도구함
- 제목·본문 통합 검색과 `Ctrl+K`·`/` 검색 단축키
- 단원별 공유 주소, 마지막 읽은 위치 복원, `←`·`→` 이전·다음 이동
- 별빛 헤더, 이동형 내비게이션 인디케이터, 절제된 전환 애니메이션
